Advanced Reporting Application

Last Updated On:
April 25, 2021
Author:
Qaisar Khan
Client:
Finsure Finance and Insurance

Commission Transactions Management System


Developed a reporting application for easy extraction or scheduling of the custom reports. Reports are sub-categorized in to number of Report Types:

  • Pipeline Reports
  • Marketing Reports
  • Referrer Reports
  • Commission Reports
  • Miscellaneous Reports
  • Custom Reports>


    • Screen Dump of the Reports main section


    Report Selection - Display data by default

    • When the user selects a report, it displays the data for the current month. If no data available, will display the message ‘No data available in table’
    • User selects the date range to display the required information
    • User can export the data as EXCEL/CSV format.
    • Print functionality is also available for the user.
    • User will have the ability to change number of columns in the report and extract the data for specific columns.
    • To restore the complete list of the columns, user could click on Restore Default and all the de-selected columns will reappear.
    • User after selecting the columns and date range can save the report for future use.
    • Click on the button “Save/Schedule”, when the user is ready to save the specific report with selected columns.
    • The system will save the custom report under the sub category “Custom Reports”
    • User can then choose this report from the Custom Reports section of the new reporting module.
